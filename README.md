# CRACKSHASH MOD • DECEPTICON — Website

## Project Overview
Ultra-professional single-page cyberpunk hacker website promoting **Decepticon — Autonomous Hacking Agent** by **Crackshash Mod**.

## Live Entry Point
- **Main URL:** `index.html`

## Features Implemented

### Design & Aesthetic
- [x] Dark cyberpunk theme (`#0a0a0f` near-black background)
- [x] Neon palette: green `#00ff9f`, red `#ff0033`, purple `#9f00ff`, blue `#00b8ff`
- [x] Matrix digital rain background (canvas, low opacity)
- [x] Scanline CSS overlay effect
- [x] Glass-morphism holographic UI panels (`backdrop-filter: blur`)
- [x] CSS glitch animation on DECEPTICON hero title (clip-path distortion + text-shadow flicker)
- [x] Pulsating neon glow on buttons and borders
- [x] Floating/bobbing animations on terminal and cards
- [x] Scroll-triggered reveal animations (Intersection Observer API)
- [x] Google Fonts: Orbitron (headings), JetBrains Mono (code/mono), Inter (body)
- [x] Font Awesome 6.4 icons for socials and UI elements

### Navigation
- [x] Fixed glass-morphism navbar with blur
- [x] Logo: "CRACKSHASH MOD • DECEPTICON" with neon flicker animation
- [x] Menu links (smooth scroll): Mission, Capabilities, Deploy, Live Demo, Intel
- [x] All 5 social media icons (YouTube, Telegram, Instagram, Facebook, WhatsApp)
- [x] "Get Source" button → GitHub
- [x] Mobile hamburger menu with full-screen overlay
- [x] Active section highlight on nav links
- [x] Navbar opacity change on scroll

### Hero Section
- [x] THREE.JS 3D scene: rotating wireframe cubes, pulsating red core sphere, orbit rings, holographic panels, particle field, grid floor
- [x] Mouse parallax effect on 3D camera
- [x] Giant glitchy "DECEPTICON" title with CSS glitch animation
- [x] Subtitle, tagline, creator credit
- [x] Three CTA buttons: Download Now, Get ZIP from MediaFire, Launch Demo
- [x] Animated terminal showing fake live kill chain output (typing effect, looping)

### Sections
- [x] **Mission** — About autonomous hacking, quote box, stats bar, operations framework cards (RoE/ConOps/Deconfliction/OPPLAN), 3-step vision, Offensive Vaccine metaphor
- [x] **Capabilities (Why Decepticon)** — 4 principle cards with hover glow (Real Red Teaming, Interactive Shells, Complete Isolation, CLI-First)
- [x] **Features** — 6 feature cards grid (Kill Chain, C2/Sliver, MITRE ATT&CK, Multi-Model Routing, Skill System, Engagement Planning)
- [x] **Architecture** — CSS network diagram (decepticon-net vs sandbox-net), 5 specialist agents display
- [x] **Deploy** — Beautiful terminal code blocks with copy buttons (install, configure, demo, dev setup)
- [x] **Live Demo** — Detailed demo terminal showing full kill chain with timestamps, multi-model routing table
- [x] **Intel** — 4 resource cards (Docs, GitHub, Discord, PurpleLab)
- [x] **Disclaimer** — Legal warning box with red border
- [x] **Footer** — Logo, all socials, copyright, Apache 2.0 badge

### Technical
- [x] THREE.js r128 via CDN
- [x] Font Awesome 6.4.0 via CDN
- [x] Google Fonts via CDN
- [x] Single HTML file — all CSS and JS inline
- [x] Fully responsive (mobile hamburger, stacked layouts at 900px and 600px breakpoints)
- [x] Copy-to-clipboard code blocks
- [x] Intersection Observer scroll reveals
- [x] requestAnimationFrame for Three.js render loop
- [x] Passive scroll event listeners
- [x] CSS custom properties for all neon colors
